Support

Last updated: May 11, 2026

LinkDish is designed to keep recipe extraction simple and dependable. If something is not working as expected, the fastest first checks are:

  • Make sure the recipe link is public and still available.
  • Try pasting the link again if the source was temporarily unreachable.
  • Try a different recipe source if the site blocks extraction or is unsupported.

Saved recipes

Saved recipes are stored on your device. If a saved recipe looks outdated or malformed after an app update, removing it and extracting it again should refresh the content.

Accounts and Family households

No account is required to import and save recipes on one device. Sign in is required only to create, join, or manage LinkDish Family household sharing.

Privacy

For details about what information the app handles, read the LinkDish Privacy Policy.

Contact

For support or privacy questions, email robertgordon761@gmail.com. If you are using the web app, include the browser and device you are using so the issue is easier to trace.

Data deletion requests

If you created a LinkDish account, you can delete it from Account in the app. Deleting the account removes server-side account and household records, but paid subscriptions still need to be managed through the checkout or store account used to subscribe. If you cannot access the app, send your request to robertgordon761@gmail.com.
